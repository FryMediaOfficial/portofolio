# FryMediaOfficial Assets

## Logo

Place your `logo.png` file in this directory.

### Requirements:
- Format: PNG (with transparency recommended)
- Recommended size: 512x512 pixels
- The logo will be used for:
  - Navbar logo (40x40px)
  - Loading screen (120x120px)
  - Favicon
  - Open Graph image

### Note:
Replace the placeholder logo.png with your actual FryMediaOfficial logo.
